Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM:

  • document.getElementById() - Selects an element by its ID.
  • document.getElementsByClassName() - Selects elements by their class name.
  • document.getElementsByTagName() - Selects elements by their tag name.
  • document.querySelector() - Selects the first element that matches a CSS selector.
  • document.querySelectorAll() - Selects all elements that match a CSS selector.
Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML returns the HTML content of an element, including all child elements and their attributes. innerText returns the visible text content of an element, excluding any HTML tags. textContent returns the text content of an element and all its descendants, including hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where a single event listener is added to a parent element instead of adding individual listeners to each child element. This approach takes advantage of event bubbling, allowing you to manage events more efficiently and reduce memory usage.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in the DOM where an event starts from the target element and bubbles up to its ancestors in the DOM tree. This means that when an event occurs on a child element, it is first captured and handled by the child, and then propagated to its parent elements, allowing them to respond to the event as well. This behavior enables event delegation and simplifies event management.

Question-5: How do you create, add, and remove elements using JavaScript?

To create a new element, you can use document.createElement(). To add an element to the DOM, you can use parentElement.appendChild() or parentElement.insertBefore(). To remove an element, you can use element.remove() or parentElement.removeChild().
